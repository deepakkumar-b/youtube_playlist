-----------------------------------------------------------------------------------------------------------------------------------
Instructions on how to run the application
-----------------------------------------------------------------------------------------------------------------------------------

1) Install npm, node, angular, bootstrap, express

2) Install nodemon to automatically poll and find for changes (eliminates time to rerun server each time)

3) Type the following command from the location where server.js is placed: nodemon server.js

4) The port assigned to the app is 1337(LEET). Go to localhost:1337

5) Choose an artist from the dropdown and see his playlist
